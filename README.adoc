= CCXS

C语言加密货币交换库。

是一个连接到websocket的加密货币交换的C语言库。

它使用<libwebsocket>作为客户端连接并转换订单簿。

.支持加密货币交易平台

== 如何用 Docker
. `cd docker-build`
. `docker build -t my-gcc .`
. `docker run -it --rm -v "$PWD/..":/usr/src/app -p 7681:7681 my-gcc`
. 内 Docker:
.. `mkdir buildbox && cd buildbox`
.. `cmake ..`
.. `make`
.. `ctest`
.. `./src/ccxs`


== 如何用
